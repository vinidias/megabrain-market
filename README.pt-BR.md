## O Que Faz

O MegaBrain Market transforma consciência situacional global em vantagem competitiva de negócios. Enquanto monitores tradicionais focam pesadamente em conflitos geopolíticos — que representam apenas uma das muitas perturbações operacionais — o MegaBrain Market expande sua visibilidade para inteligência preditiva de mercado, corredores comerciais, commodities e transição energética:

- **Sinal de Mercado e Negócios Proativo** — 500+ feeds de inteligência curados em finanças, commodities, transição energética, tecnologia e geopolítica, sintetizados por IA em briefings executivos acionáveis
- **Corredores de Energia Regional e Hidrogênio Verde** — rastreamento especializado de megaprojetos de energia limpa, complexos eólicos/solares e polos portuários industriais (incluindo cobertura profunda do hub de Hidrogênio Verde no Ceará e corredor renovável do Nordeste brasileiro)
- **Monitoramento de Cadeia de Suprimentos e Rotas Comerciais** — rastreamento em tempo real dos 30+ maiores portos de contêineres, terminais de petróleo/GNL e gargalos estratégicos para blindar operações e logística empresarial
- **Motor de Visualização de Mapa Duplo** — globo 3D (globe.gl) e mapa plano WebGL (deck.gl) com 56 tipos de camadas para visualização geográfica de atividade econômica e riscos
- **Radar Financeiro e de Commodities** — 29 bolsas de valores, commodities agrícolas e energéticas, cripto e índice composto de mercado de 7 sinais com alertas em tempo real
- **Correlação entre Fluxos** — convergência de indicadores econômicos, mudanças de política comercial, infraestrutura e sinais ambientais para análise preditiva de mercado
- **IA Local e Soberana** — rode tudo no seu ambiente com Ollama, sem necessidade de chaves de API externas — total privacidade e soberania sobre seus dados
- **6 Variantes de Site Especializadas** a partir de uma única base de código (`market.`, `tech.`, `finance.`, `commodity.`, `energy.`, `risk.`)
- **App Desktop Nativo** (Tauri 2) para macOS, Windows e Linux, com suporte a 25 idiomas

Para a lista completa de funcionalidades, arquitetura, fontes de dados e algoritmos, veja a **[documentação](https://megabrain.market/docs/documentation)**.

## Início Rápido

```bash
git clone https://github.com/vinidias/megabrain-market.git
cd megabrain-market
npm install
npm run dev
```

Abra [localhost:3000](http://localhost:3000) (altere a porta com `DEV_PORT` no `.env.local`). O app roda sem nenhuma variável de ambiente.

Fontes de dados específicas de funcionalidades podem exigir credenciais. Veja `.env.example` para a lista completa.

Para desenvolvimento de variantes específicas:

```bash
npm run dev:tech       # tech.megabrain.market
npm run dev:finance    # finance.megabrain.market
npm run dev:commodity  # commodity.megabrain.market
npm run dev:energy     # energy.megabrain.market
npm run dev:risk       # risk.megabrain.market
```

Veja o **[guia de self-hosting](https://megabrain.market/docs/getting-started)** para opções de deploy (Vercel, Docker, estático).

## Stack Tecnológica

| Categoria | Tecnologias |
|-----------|-------------|
| **Frontend** | TypeScript Vanilla, Vite, globe.gl + Three.js, deck.gl + MapLibre GL |
| **Desktop** | Tauri 2 (Rust) com sidecar Node.js |
| **IA/ML** | Ollama / Groq / OpenRouter, Transformers.js (lado do navegador) |
| **Contratos de API** | Protocol Buffers (281 protos, 35 serviços), anotações HTTP sebuf |
| **Deploy** | Vercel Edge Functions (60+), Railway relay, Tauri, PWA |
| **Cache** | Redis (Upstash), cache de 3 camadas, CDN, service worker |

Detalhes completos da stack na **[documentação de arquitetura](https://megabrain.market/docs/architecture)**.

## Acesso Programático

O MegaBrain Market é construído tanto para agentes e scripts quanto para navegadores:

- **Servidor MCP** — `https://megabrain.market/mcp` (HTTP Streamable). `tools/list` público; `tools/call` autentica com header `X-MegaBrain-Key` ou OAuth.
- **API REST** — base `https://api.megabrain.market`, descrita pela [especificação OpenAPI](https://megabrain.market/openapi.yaml).
- **CLI** — o pacote npm oficial [`megabrain-market`](https://www.npmjs.com/package/megabrain-market) (código em [`cli/`](cli/)):

  ```sh
  npx megabrain-market tools          # executa ad-hoc — lista todas as ferramentas MCP (sem chave)
  npm install -g megabrain-market     # ou instale o comando `megabrain-market` (alias `mb`)
  megabrain-market risk BR --api-key mb_xxx
  ```

- **SDKs** — bibliotecas cliente oficiais zero-dependência espelhando a CLI: Python [`megabrain-market-sdk`](https://pypi.org/project/megabrain-market-sdk/) (código em [`sdk/python/`](sdk/python/)), Go [`github.com/vinidias/megabrain-market/sdk/go`](https://pkg.go.dev/github.com/vinidias/megabrain-market/sdk/go) ([`sdk/go/`](sdk/go/)). Guia: [megabrain.market/docs/sdks](https://megabrain.market/docs/sdks).

Arquivos de descoberta para agentes: [`llms.txt`](https://megabrain.market/llms.txt) · [manifesto agent-skills](https://megabrain.market/.well-known/agent-skills/index.json) · [api-catalog](https://megabrain.market/.well-known/api-catalog). Obtenha uma chave de API em [megabrain.market/pro](https://megabrain.market/pro).
