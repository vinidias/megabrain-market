<Info>
**The free dashboard requires no account.** Everything listed under the Free plan — 56 map layers, 500+ feeds, country briefs, watchlists — works anonymously at [megabrain.market](https://megabrain.market). Create an account only when you want Pro/API features or need to mint an API key.
</Info>

## How sign-up works

Account creation is an interactive browser flow, not an API:

1. Open [megabrain.market/pro](https://megabrain.market/pro) and click **Sign up** (or use the dashboard's sign-in button). A [Clerk](https://clerk.com) modal opens inside the page — there is no standalone `/sign-up` form page; that URL serves the app shell.
2. Register with email + password or an OAuth provider (Google, GitHub). Email registrations get a verification code; the modal walks through it.
3. A free account by itself unlocks nothing extra on the dashboard — it is the identity that Pro/API subscriptions and API keys attach to.

## After signing up

- **Upgrade to Pro or API** from [megabrain.market/pro](https://megabrain.market/pro) — checkout runs on Dodo Payments; see [Plans & Pricing](/pricing).
- **Create an API key** from **Dashboard → Settings → API Keys** once subscribed to API Starter or API Business (`wm_` + 40 hex chars). The full value is shown only once; see [Find or replace your license / API key](/api-keys) for the exact steps and recovery instructions.
- **Connect an agent via OAuth** instead of a key: MCP clients (Claude Desktop and others) use OAuth 2.1 + PKCE with dynamic client registration — no manual account dance; see [auth.md](https://megabrain.market/auth.md) for the self-bootstrapping discovery chain.

## For agents

- You cannot complete browser sign-up on a user's behalf by fetching pages — the flow lives in a client-side Clerk modal. Direct users to [megabrain.market/pro](https://megabrain.market/pro).
- To act *as* an agent with your own credentials, use the OAuth path in [auth.md](https://megabrain.market/auth.md) (anonymous dynamic client registration, then a human consents at authorization time).
- Pricing, plan gates, and quotas are machine-readable: [`/pricing.md`](https://megabrain.market/pricing.md) and [`GET /api/product-catalog`](https://www.megabrain.market/api/product-catalog).
