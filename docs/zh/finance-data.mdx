## 市场监控

### 可自定义的市场自选列表

市场(Markets)面板支持用户自定义的股票和商品代码列表,允许分析师跟踪默认指数集合之外的特定品种。自选列表接受多种格式:

- **指数代码**:`^GSPC`、`^DJI`、`^IXIC`
- **股票代码**:`AAPL`、`BRK-B`、`NVDA`
- **大宗商品**:`GC=F`(黄金)、`CL=F`(原油)
- **加密货币对**:`BTC-USD`、`ETH-USD`
- **友好标签**:`TSLA|Tesla Inc`(用管道符分隔的显示名称)

代码通过市场面板设置图标可访问的模态对话框输入。输入接受逗号分隔、换行分隔或 `SYMBOL|Label` 格式。自选列表状态存储在 `localStorage` 中,键名为 `wm-market-watchlist-v1`(最多 50 个代码,自动去重)。变更会发出 `wm-market-watchlist-changed` CustomEvent,用于在面板之间实时同步而无需刷新页面。

### 宏观信号分析(市场雷达)

市场雷达面板根据 7 个独立信号计算出一个综合的 BUY/CASH 判定,这些信号全部来源于免费 API(Yahoo Finance、mempool.space、alternative.me):

| 信号               | 计算方式                               | 看涨条件                     |
| ------------------- | ------------------------------------- | --------------------------- |
| **流动性**          | JPY/USD 30 日变化率                    | ROC > -2%(无日元挤兑)       |
| **资金流向结构**    | BTC 5 日回报 vs QQQ 5 日回报           | 差距 &lt; 5%(同步)           |
| **宏观格局**        | QQQ 20 日 ROC vs XLP 20 日 ROC         | QQQ 表现更优(风险偏好)      |
| **技术趋势**        | BTC vs SMA50 + 30 日 VWAP             | 两者之上(看涨)              |
| **算力**            | 比特币挖矿算力 30 日变化               | 增长 > 3%                   |
| **挖矿成本**        | BTC 价格 vs 算力隐含成本               | 价格 > $60K(盈利)          |
| **恐慌与贪婪**      | alternative.me 情绪指数                | 值 > 50                     |

整体判定要求已知信号中 ≥57% 为看涨(BUY),否则为 CASH。数据未知的信号不计入分母。

**VWAP 计算** —— 成交量加权平均价(Volume-Weighted Average Price)由 30 天窗口内对齐的价格/成交量配对计算得出。价格或成交量为 null 的配对会被一起排除,以防止指数错位:

```
VWAP = Σ(price × volume) / Σ(volume)    for last 30 trading days
```

**Mayer 倍数**(BTC 价格 / SMA200)提供了长期估值背景 —— 历史上,数值高于 2.4 表示过热,而低于 0.8 则暗示严重低估。

Finance 变体包含一个精选数据库,记录了沙特阿拉伯和阿联酋在全球关键基础设施领域的 64 项重大外商直接投资。这些投资覆盖 12 个行业:

| 行业              | 示例                                                                                                 |
| ----------------- | ---------------------------------------------------------------------------------------------------- |

| **能源**          | ADNOC Ruwais LNG(9.6 mtpa)、Aramco 的 Motiva Port Arthur 炼油厂(630K bpd)、ACWA Power 可再生能源 |
| **制造业**        | Mubadala 的 GlobalFoundries(82% 股份,第 3 大芯片代工厂)、Borealis(75%)、SABIC(70%)               |

| **超级项目**      | NEOM THE LINE(5000 亿美元)、沙特国家云(60 亿美元超大规模数据中心)                                   |

每项投资记录了投资主体(DP World、Mubadala、PIF、ADNOC、Masdar、Saudi Aramco、ACWA Power 等)、目标国家、地理坐标、投资金额(美元)、持股比例、运营状态和年份。投资面板提供按国家(SA/UAE)、行业、主体和状态的可筛选视图 —— 点击任意行会将地图导航到投资位置。

在地球上,投资以缩放气泡的形式显示:≥500 亿美元的项目(NEOM)以最大尺寸渲染,而低于 10 亿美元的投资使用较小的标记。颜色编码状态:绿色表示运营中,琥珀色表示建设中,蓝色表示已宣布。

## 加密货币与稳定币

### 稳定币锚定监控

通过 CoinGecko API 监控五种主要稳定币(USDT、USDC、DAI、FDUSD、USDe),缓存时间为 2 分钟。每种币相对于 1.00 美元锚定的偏离程度决定其健康状态:

| 偏离程度     | 状态         | 指示器     |
| ----------- | ------------ | --------- |
| ≤ 0.5%      | ON PEG       | 绿色       |
| 0.5% – 1.0% | SLIGHT DEPEG | 黄色       |
| > 1.0%      | DEPEGGED     | 红色       |

面板聚合稳定币总市值、24 小时成交量和整体健康状态(HEALTHY / CAUTION / WARNING)。`coins` 查询参数接受逗号分隔的 CoinGecko ID 列表,通过 `[a-z0-9-]+` 正则验证以防止注入。

### BTC ETF 资金流向估算

通过 Yahoo Finance 的 5 日图表 API 跟踪十只现货比特币 ETF(IBIT、FBTC、ARKB、BITB、GBTC、HODL、BRRR、EZBC、BTCO、BTCW)。由于 ETF 资金流向数据需要昂贵的终端订阅,系统通过公开可用的信号估算资金流向:

- **价格变化** —— 当日收盘价与前一交易日收盘价对比决定方向
- **成交量比率** —— 当前成交量 / 滚动平均成交量,衡量市场信心
- **资金规模** —— `volume × price × direction × 0.1` 提供粗略的美元估算

这是一种近似估算,不能替代官方资金流向数据,但能正确捕捉方向和相对规模。结果缓存 15 分钟。

### 石油与能源分析

石油与能源面板通过美国能源信息署(EIA)API 跟踪四个关键指标:

| 指标              | 数据序列                  | 更新频率        |
| ----------------- | ------------------------- | -------------- |
| **WTI 原油**      | 现货价格($/桶)            | 每周            |
| **布伦特原油**    | 现货价格($/桶)            | 每周            |
| **美国产量**      | 原油产量(千桶/日)         | 每周            |
| **美国库存**      | 商业原油库存               | 每周            |

## 央行与贸易

### BIS 央行数据

经济面板整合了来自国际清算银行(BIS,央行中的央行)的数据,提供三个互补的数据集:

| 数据集            | 描述                                                       | 用途                                                      |
| ----------------- | ---------------------------------------------------------- | --------------------------------------------------------- |
| **政策利率**      | 主要经济体的当前央行政策利率                                | 货币政策立场比较 —— 紧缩 vs. 宽松                          |
| **实际有效汇率**  | 经通胀调整的贸易加权货币指数(REER)                          | 货币竞争力 —— REER 上升 = 升值,下降 = 贬值                |
| **信贷/GDP**      | 非金融部门总信贷占 GDP 的百分比                            | 信贷泡沫检测 —— 高比率意味着经济过度杠杆                  |

数据通过 `economic/v1` proto 服务中的三个专用 BIS RPC(`GetBisPolicyRates`、`GetBisExchangeRates`、`GetBisCredit`)获取。每个数据集使用独立的熔断器,缓存 TTL 为 30 分钟。面板将政策利率渲染为带有 spark bars 的排序表格,汇率带有方向性趋势指示器,信贷/GDP 渲染为排序列表。BIS 数据的新鲜度在情报缺口系统中跟踪 —— 过期或失败会以明确的警告呈现,而非静默缺口。

### WTO 贸易政策情报

贸易政策面板提供对全球贸易限制、关税和壁垒的实时可见性 —— 这对于跟踪经济战、制裁影响和供应链中断风险至关重要。提供六种数据视图:

| 标签页        | 数据源                                 | 内容                                                                                   |
| ------------- | -------------------------------------- | -------------------------------------------------------------------------------------- |
| **限制**      | WTO 贸易监控                            | 活跃的贸易限制,包含实施方/受影响国家、产品类别和执行日期                                |
| **关税**      | WTO 关税数据库                          | 国家对之间的关税税率趋势(例如 US↔China),含历史数据点                                  |
| **流量**      | WTO 贸易统计                            | 双边贸易流量,含同比变化指标                                                            |
| **壁垒**      | WTO SPS/TBT 通报                        | 卫生、植物卫生和技术性贸易壁垒,含状态跟踪                                              |
| **收入**      | 美国财政部月度财政声明                  | 每月美国海关关税收入,含财年累计总额和同比对比                                          |
| **Comtrade**  | UN Comtrade                            | 带异常检测的战略大宗商品流量搜索                                                        |

`trade/v1` proto 服务定义了六个 RPC。WTO 支撑的限制、关税、流量与壁垒 RPC 各自有独立的熔断器(30 分钟缓存 TTL)和 `upstreamUnavailable` 信号,用于在 WTO 端点暂时不可达时优雅降级。`GetCustomsRevenue` 从 Railway seed 提供美国财政部关税数据(免费 API,无需密钥),`ListComtradeFlows` 提供 UN Comtrade 战略大宗商品流量。该面板在 full/geopolitical、finance 和 commodity 变体上可用。WTO 数据以 `wto_trade` 接入数据新鲜度跟踪器,财政部收入以 `treasury_revenue` 接入,任一数据源过期时会触发情报缺口警告。关于确切的方法、路由和门控说明,请参见[贸易政策](/zh/panels/trade-policy)。

在贸易政策活跃变化期间,收入标签页尤为相关。继 2025-2026 年关税升级后,美国海关关税收入从约 70 亿美元/月(2025 年前)飙升至 270-310 亿美元/月。WTO 年度关税数据滞后约一年,因此月度财政部收入数据提供了对关税政策实际财政影响的近实时可见性。收入标签页显示财年累计摘要,使用匹配的财政月份数进行同比对比(例如 FY2026 10 月-2 月 vs FY2025 10 月-2 月),并提供月度表格,高亮显示收入超过上年月均值两倍的月份。在未配置 WTO API 密钥的桌面端,收入标签页仍可访问,因为财政部数据无需身份验证。

## 供应链

### 供应链中断情报

供应链面板跨三个互补维度提供对全球物流风险的实时可见性 —— 战略咽喉要道健康状况、运输成本趋势和关键矿产集中度 —— 从而能够及早发现会级联演变为经济和地缘政治后果的中断事件。

**咽喉要道标签页** —— 通过交叉比对实时航行警告与 AIS 船舶中断数据,监控 9 条战略水道(苏伊士运河、马六甲海峡、霍尔木兹海峡、曼德海峡、巴拿马运河、台湾海峡、直布罗陀海峡、博斯普鲁斯海峡、达达尼尔海峡)。每个咽喉要道获得一个中断评分(0–100),由三部分公式计算得出:基线威胁等级(战区 / 关键 / 高 / 较高 / 正常)、活跃警告数量(有上限贡献)和 AIS 拥堵严重程度 —— 映射为颜色编码的状态指示器(绿/黄/红)。咽喉要道识别使用文本证据匹配(以主要术语和区域术语进行关键词评分),然后才回退到地理邻近度,以防将提及某一咽喉要道但地理上更接近另一咽喉要道的事件错误分类。数据缓存 TTL 为 5 分钟,以实现近实时感知。

**运输费率标签页** —— 跟踪两个美联储经济数据(FRED)序列:深海货运生产者价格指数(`PCU483111483111`)和货运运输服务指数(`TSIFRGHT`)。统计尖峰检测将异常价格变动与近期历史进行对比标记。内联 SVG sparklines 一目了然地呈现 24 个月的费率历史。缓存 1 小时,以反映底层数据的每周发布节奏。

**关键矿产标签页** —— 将**赫芬达尔-赫希曼指数(HHI)**应用于 2024 年对科技和国防制造至关重要的矿产的全球生产数据 —— 锂、钴、稀土、镓、锗等。HHI 量化供应集中度风险:由单一生产者主导的市场得分接近 10,000,而完全分散的市场得分接近 0。每种矿产显示排名前 3 的生产国及其市场份额百分比,标记出代表战略脆弱性的单一国家依赖(例如中国在稀土加工中的主导地位)。此标签页使用静态生产数据,缓存 24 小时,无外部 API 依赖。

该面板在 full/geopolitical、finance 和 commodity 变体上可用,并与基础设施级联模型集成 —— 当咽喉要道中断与受影响贸易路线的高矿产集中度风险同时发生时,组合信号会馈入收敛检测。

另请参阅[海事情报](/zh/maritime-intelligence),了解船舶跟踪和暗船检测。
