MegaBrain Market 提供了一套全面的实时全球情报监控功能，从交互式地图和数据图层到市场追踪和可自定义的面板布局。

## 交互式全球地图

- **缩放与平移** - 使用鼠标/触控板手势进行流畅导航
- **区域聚焦** - 8 个预设视图用于快速导航（全球、美洲、欧洲、中东、亚洲、拉丁美洲、非洲、大洋洲）
- **图层系统** - 切换按类别组织的 56 个数据图层的可见性
- **时间筛选** - 按时间范围筛选事件（1h、6h、24h、48h、7d）
- **可固定地图** - 滚动浏览面板时将地图固定在顶部，或让其随页面滚动
- **智能标记聚类** - 附近的标记在低缩放级别时聚合，放大后展开

## 标记聚类

数据点密集的区域使用智能聚类以防止视觉混乱：

**工作原理**

- 在像素半径内（根据缩放级别自适应）的标记合并为聚类徽章
- 聚类徽章显示分组项目的数量
- 点击聚类会打开一个弹窗，列出所有分组项目
- 放大可减小聚类半径，最终显示单个标记

**分组逻辑**

- **抗议活动**：仅在同一国家内聚类（暴乱优先排序，高严重性优先）
- **科技总部**：在同一城市内聚类（大型科技优先于独角兽，独角兽优先于上市公司）
- **科技事件**：在同一地点内聚类（按日期排序，最近的在前）

这避免了迪拜和利雅得的公司在全球缩放级别下显示为合并状态的问题，同时在大陆尺度上仍提供清晰的视觉效果。

## 来源筛选

标题栏中的 **SOURCES** 按钮可打开全局来源管理弹窗，实现对仪表板中显示的新闻来源的精细控制。

**功能：**

- **搜索**：按名称筛选来源列表以快速查找特定媒体
- **单独切换**：点击任何来源以启用/禁用
- **批量操作**：使用"全选"和"全不选"快速调整
- **计数显示**：显示"45/77 已启用"以指示当前选择
- **持久化**：设置保存到 localStorage 并跨会话保留

**使用场景：**

- **降噪**：禁用高产量的聚合器（Google News）以专注于主要来源
- **区域聚焦**：仅启用与特定地理区域相关的来源
- **来源质量**：禁用信噪比较差的来源
- **偏见管理**：通过启用/禁用具有已知编辑立场的来源来平衡覆盖

**技术细节：**

- 禁用的来源在获取时（而非显示时）被过滤，减少带宽和 API 调用
- 同时影响所有新闻面板：禁用 BBC 一次，所有地方都不再显示
- 所有来源被禁用的面板显示"所有来源已禁用"消息
- 更改在下次刷新周期生效

## 实时新闻流

嵌入的 YouTube 主要新闻网直播流，支持频道切换：

| 频道 | 覆盖范围 |
|---------|----------|
| **Bloomberg** | 商业与金融新闻 |
| **Sky News** | 英国与国际新闻 |
| **Euronews** | 欧洲视角 |
| **DW News** | 德国国际广播 |
| **France 24** | 法国全球新闻 |
| **Al Arabiya** | 中东新闻（阿拉伯视角） |
| **Al Jazeera** | 中东与国际新闻 |

**核心功能：**

- **频道切换器** - 一键切换网络
- **直播指示器** - 闪烁的圆点显示流状态，点击暂停/播放
- **静音切换** - 音频控制（默认静音）
- **双倍宽度面板** - 更大的视频播放器以获得更好的观看体验

**性能优化：**

直播流面板使用 **YouTube IFrame Player API** 而非原始 iframe 嵌入。这提供了几个优势：

| 功能 | 优势 |
|---------|---------|
| **持久播放器** | 静音/播放/频道切换时无 iframe 重载 |
| **API 控制** | 直接调用 `playVideo()`、`pauseVideo()`、`mute()` |
| **减少带宽** | 状态变化时同一流继续播放 |
| **更快切换** | 通过 `loadVideoById()` 切换频道 |

**空闲检测：**

为节省资源，面板实现了自动空闲暂停：

| 触发条件 | 动作 |
|---------|--------|
| **标签页隐藏** | 流暂停（通过 Visibility API） |
| **空闲 5 分钟** | 流暂停（无鼠标/键盘活动） |
| **用户返回** | 流自动恢复 |
| **手动暂停** | 用户意图单独追踪 |

这防止了后台标签页消耗带宽，同时保留了用户对手动暂停流的偏好。

## 工作流

两个全屏的、键盘优先的工作流补充了地图 + 面板体验：

- **[Route Explorer](/zh/route-explorer)** — 规划任意两个国家之间的货运。四个标签页（当前 / 替代方案 / 陆运 / 影响）解析所选 HS2 商品的咽喉点暴露、绕行通道、陆运替代方案和各国进口暴露。PRO 限制。
- **[Scenario Engine](/zh/scenario-engine)** — 运行预置的中断场景（冲突、天气、制裁、关税冲击），在地图和供应链面板中观察影响在咽喉点、板块和国家间的解析。PRO 限制。

## 国家级风险

两个互补的国家评分系统：

- **[国家不稳定指数](/zh/country-instability-index)** — 高频 CII v8 压力评分，含带符号的 24 小时变动增量，融合 31 个一级国家的动荡、冲突、安全和信息信号。
- **[国家韧性指数](/zh/methodology/country-resilience-index)** — 针对涵盖 6 个领域和 20 个活跃维度（归入 3 大支柱：结构准备度、实时冲击暴露、恢复能力；另含 2 个为保持 schema 连续性而保留的退役维度）的 196 国公开可排序范围的 0-100 韧性评分，每 6 小时刷新，来源于官方/权威数据提供方，含透明的插补。

CII 和 CRI 回答不同的问题，不可互换：CII 是短期的压力；CRI 是结构韧性加实时冲击暴露。

仪表板高亮新到达的项目，以便您可以快速识别自上次查看以来发生了什么变化。

### 视觉指示器

| 指示器 | 持续时间 | 用途 |
|-----------|----------|---------|
| **NEW 标签** | 2 分钟 | 刚出现的项目的徽章 |
| **发光高亮** | 30 秒 | 引起注意的微妙动画 |
| **面板徽章** | 直到查看 | 折叠面板中新项目的计数 |

### 自动"已读"检测

系统使用 IntersectionObserver 检测面板何时变为可见：

- 当面板 >50% 可见超过 500ms 时，项目被标记为"已读"
- 滚动浏览面板时渐进标记可见项目
- 切换面板时适当重置"新"状态

### 面板专属追踪

每个面板维护独立的活动状态：

- **新闻**：自上次查看以来的新聚类
- **市场**：超过阈值的价格变化
- **预测**：概率变化 >5%
- **自然事件**：新地震和 EONET 事件

这实现了聚焦监控：您可以折叠已审阅的面板，一目了然地查看哪些面板有新活动。

## 社会动荡追踪

抗议图层聚合来自两个独立来源的国内动荡数据，提供相互印证和全球覆盖。

### ACLED（武装冲突地点与事件数据）

学术级冲突数据，含人工核实的事件：

- **覆盖**：全球，30 天滚动窗口
- **事件类型**：抗议、暴乱、罢工、示威
- **元数据**：参与方、伤亡、详细备注
- **置信度**：高（人工整理）

### GDELT（全球事件、语言和语调数据库）

实时新闻衍生事件数据：

- **覆盖**：全球，7 天滚动窗口
- **事件类型**：来自新闻的地理编码抗议提及
- **数量**：每个地点的报道数（信号强度）
- **置信度**：中（算法提取）

### 多源印证

来自两个来源的事件使用 0.5 度空间网格和日期匹配进行去重。当 ACLED 和 GDELT 都在同一区域报告事件时：

- 置信度提升为"高"
- ACLED 数据优先（精度更高）
- 来源列表显示印证

### 严重性分类

| 严重性 | 标准 |
|----------|----------|
| **高** | 有伤亡报告、暴乱或冲突 |
| **中** | 大型示威、罢工 |
| **低** | 较小规模抗议、局部事件 |

情报热点附近的事件会被交叉引用以提供地缘政治背景。

### 地图显示筛选

为减少视觉混乱并聚焦重要事件，地图仅显示**高严重性抗议和暴乱**：

| 显示 | 事件类型 | 视觉 |
|-----------|------------|--------|
| 是 | 暴乱 | 亮红色标记 |
| 是 | 高严重性抗议 | 红色标记 |
| 否 | 中/低严重性抗议 | 地图上不显示 |

较低严重性的事件仍被追踪用于 CII 评分和数据导出：它们只是不会创建地图标记。此筛选可防止密集城区（自然会生成更多低严重性示威）淹没地图显示。

## 航空监控

航班图层使用 FAA NASSTATUS 数据追踪美国主要机场的机场延误和地面停飞。

### 延误类型

| 类型 | 描述 |
|------|-------------|
| **地面停飞** | 不允许起飞；严重中断 |
| **地面延误** | 起飞被扣留；到达率受限 |
| **到达延误** | 进港流量积压 |
| **出发延误** | 出港流量延误 |

### 严重性阈值

| 严重性 | 平均延误 | 视觉 |
|----------|--------------|--------|
| **严重** | ≥60 分钟 | 红色 |
| **重大** | 45-59 分钟 | 橙色 |
| **中等** | 25-44 分钟 | 黄色 |
| **轻微** | 15-24 分钟 | 灰色 |

### 监控机场

通过三个独立数据源（FAA ASWS、AviationStack、ICAO NOTAM）监控跨 5 个区域（美洲、欧洲、亚太、MENA、非洲）的 111 个机场：

- 美国主要枢纽：JFK、LAX、ORD、ATL、DFW、DEN、SFO
- 具有高客流量的国际门户
- 通过 ICAO NOTAM 检测关闭的 46 个 MENA 机场
- 经常受天气或拥堵影响的机场

地面停飞尤为重要：它们表明严重中断（天气、安全或基础设施故障），并可能在网络中级联传播。

## 面板管理

仪表板将数据组织为**可拖拽、可折叠的面板**，跨会话保留用户偏好。

### 拖拽重排

面板可以通过拖拽重新组织：

1. 抓取面板标题栏（悬停时出现抓手图标）
2. 拖拽到所需位置
3. 释放以重新排序
4. 新顺序自动保存到 LocalStorage

这实现了个性化布局：将您最关注的面板放在顶部。

### 面板可见性

通过设置菜单切换面板开/关：

- **隐藏面板**：不渲染，不获取数据
- **可见面板**：完整功能
- **折叠面板**：仅显示标题栏，数据仍刷新

隐藏面板与禁用图层不同：面板本身不会出现在界面中。

### 默认面板顺序

面板按情报优先级组织：

| 优先级 | 面板 | 用途 |
|----------|--------|---------|
| **关键** | 战略风险、实时情报 | 即时态势感知 |
| **主要** | 新闻、CII、市场 | 核心监控数据 |
| **辅助** | 预测、经济、监控器 | 补充分析 |
| **参考** | 实时新闻视频 | 背景信息 |

### 持久化

面板状态在浏览器重启后仍然保留：

- **LocalStorage**：面板顺序、可见性、折叠状态
- **自动保存**：更改立即持久化
- **按设备**：设置为浏览器专属（不同步）

## 移动端体验

仪表板针对移动设备进行了优化，提供精简界面，在小屏幕上优先考虑可用性。

### 首次移动端欢迎

首次在移动设备上访问仪表板时，欢迎弹窗会说明移动端优化体验：

- **精简视图提示** - 告知用户正在查看精选的移动版本
- **导航提示** - 说明区域视图按钮和标记交互
- **"不再显示"选项** - 复选框可在以后访问时跳过（持久化到 localStorage）

### 移动优先设计

在窄于 768px 的屏幕或触控设备上：

- **紧凑地图** - 降低高度（40vh）以显示更多面板
- **单列布局** - 面板垂直堆叠以便滚动
- **隐藏地图标签** - 所有标记标签隐藏以减少视觉混乱
- **固定图层集** - 图层切换按钮隐藏；默认启用精选图层集
- **简化控制** - 地图调整大小手柄和固定按钮隐藏
- **触控优化标记** - 扩大的触控目标（44px）便于点击
- **隐藏 DEFCON 指示器** - 五角大楼披萨指数隐藏以减少标题栏混乱
- **隐藏 FOCUS 选择器** - 区域聚焦按钮隐藏（改用预设视图）
- **紧凑标题栏** - 社交链接显示 X logo 而非用户名文本

### 移动端默认图层

移动端体验聚焦于最核心的情报图层：

| 图层 | 用途 |
|-------|---------|
| **冲突** | 当前冲突区域 |
| **热点** | 含活动级别的情报热点 |
| **制裁** | 处于经济制裁下的国家 |
| **中断** | 网络中断 |
| **自然** | 地震、风暴、野火 |
| **天气** | 严重天气警报 |

移动端默认禁用（但桌面端可用）的图层：

- 军事基地、核设施、航天发射场、矿产
- 海底光缆、管道、数据中心
- AIS 船舶、军事航班
- 抗议、经济中心

这一精选集合提供态势感知，同时不会使界面不堪重负或消耗过多数据/电量。

## 离线 ML 能力

仪表板包含无需任何服务器连接即可运行的浏览器端机器学习：

- **威胁分类** - 三阶段流水线（关键词预筛选、浏览器 ML 模型、可选 LLM 精炼）按威胁类别对新闻标题进行分类
- **标题评分** - 基于 ML 的新闻文章重要性评分，支持基于优先级的渲染
- **实体提取** - 客户端命名实体识别，用于识别标题中的国家、组织和关键人物

这些模型完全通过 Web Workers 在浏览器中运行，即使在离线或未配置 API 密钥时也能提供情报分析能力。

## 使用

### 键盘快捷键

- `Cmd+K` / `Ctrl+K` - 打开搜索
- `Up/Down` - 导航搜索结果
- `Enter` - 选择结果

### 地图控制

- **滚动** - 缩放
- **拖拽** - 平移地图
- **点击标记** - 显示含完整上下文的详细弹窗
- **悬停标记** - 显示含摘要信息的工具提示
- **图层切换** - 显示/隐藏数据图层
- **图层说明** - 从图层选择器打开紧凑卡片，查看图层的用途、来源/提供方、新鲜度预期、置信度说明、局限性与相关面板

### 地图标记设计

基础设施标记（核设施、经济中心、港口）显示时无标签以减少视觉混乱。完整信息可通过交互获取：

| 图层 | 标签行为 | 交互 |
|-------|---------------|-------------|
| 核设施 | 隐藏 | 点击获取含详情的弹窗 |
| 经济中心 | 隐藏 | 点击获取含详情的弹窗 |
| 抗议 | 隐藏 | 悬停获取工具提示，点击获取详情 |
| 军事基地 | 隐藏 | 点击获取含基地信息的弹窗 |
| 热点 | 可见 | 颜色编码的活动级别 |
| 冲突 | 可见 | 状态和参与方 |

此设计优先考虑地理感知而非标签密度：用户可以快速扫描标记，然后通过交互获取上下文。

### 面板管理

- **拖拽面板** - 重新排序布局
- **设置** - 切换面板可见性

### 可分享链接

当前视图状态被编码到 URL 中，支持：

- **书签**：保存特定视图以便快速访问
- **分享**：向同事发送指向您精确地图位置和图层配置的链接
- **深度链接**：直接链接到特定区域或功能

**编码参数**：
| 参数 | 描述 |
|-----------|-------------|
| `lat`, `lon` | 地图中心坐标 |
| `zoom` | 缩放级别 (1-10) |
| `time` | 活动时间筛选 (1h, 6h, 24h, 7d) |
| `view` | 预设视图 (global, us, mena) |
| `layers` | 逗号分隔的已启用图层 ID |

示例：`?lat=38.9&lon=-77&zoom=6&layers=bases,conflicts,hotspots`

数值经过验证和钳制以防止无效状态。
