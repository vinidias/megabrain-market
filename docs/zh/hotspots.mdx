MegaBrainMarket 使用实时新闻关联来检测和可视化全球热点，提供用于快速上下文切换的区域焦点预设，并支持用于持续监控工作流的地图固定功能。

## 动态热点活动

地图上的热点**不是静态威胁等级**。活动是基于新闻关联实时计算的。

每个热点定义了关键词：
```typescript
{
  id: 'dc',
  name: 'DC',
  keywords: ['pentagon', 'white house', 'congress', 'cia', 'nsa', ...],
  agencies: ['Pentagon', 'CIA', 'NSA', 'State Dept'],
}
```

系统统计当前订阅源中匹配的新闻文章，应用速度分析，并分配活动等级：

| 等级 | 标准 | 视觉表现 |
|-------|----------|--------|
| **低** | &lt;3 条匹配，正常速度 | 灰色标记 |
| **升高** | 3-6 条匹配或速度升高 | 黄色脉冲 |
| **高** | >6 条匹配或速度飙升 | 红色脉冲 |

这创建了一个基于实时新闻流的全球关注度动态"热力图"。

### 热点升级信号

除了视觉活动等级外，当热点在多个维度上显示显著变化时，系统会生成**升级信号**。这种多组件方法通过要求来自独立数据流的佐证来减少误报。

**升级组件**

每个热点的升级分数融合了四个加权组件：

| 组件 | 权重 | 数据源 | 衡量内容 |
|-----------|--------|-------------|------------------|
| **新闻活动** | 35% | RSS 订阅源 | 匹配的新闻计数、突发标志、速度 |
| **CII 贡献** | 25% | 国家不稳定指数 | 关联国家的不稳定分数 |
| **地理收敛** | 25% | 多源事件 | 地理单元中的事件类型多样性 |
| **军事活动** | 15% | OpenSky/AIS | 200公里范围内的航班和船只 |

**分数计算**

```
static_baseline = hotspot.escalationScore ?? 3  // 1-5 per hotspot
dynamic_raw = (
  news_component × 0.35 +
  cii_component × 0.25 +
  geo_component × 0.25 +
  military_component × 0.15
)

dynamic_score = 1 + (dynamic_raw / 100) × 4  // maps 0-100 inputs onto 1-5
final_score = static_baseline × 0.30 + dynamic_score × 0.70
```

静态基线是每个热点配置上的编辑性 `escalationScore`（缺失时默认为 `3`）。它刻意保持在 1-5 的同一标度上，与动态输出一致；弹窗将其作为 `staticBaseline` 与各组件柱状图一同展示，便于用户区分结构性风险与实时变动。许多热点还带有 `whyItMatters` 叙述性理由供弹窗展示，但该文本不影响评分。代码不会对最终分数施加地理邻近度乘数。

**趋势检测**

系统为每个热点维护 48 点历史（24 小时，每 30 分钟一次）：

- **线性回归**计算近期分数的斜率
- **`escalating`**：斜率 > +0.1 点/间隔
- **`de-escalating`**：斜率 &lt; -0.1 点/间隔
- **`stable`**：斜率在 ±0.1 以内、有效历史点少于 3 个，或回归分母为零

**信号生成**

当满足以下条件时，会发出升级信号（`hotspot_escalation`）：

1. 某热点跨入更高整数分段的分数带，且达到或超过 `2/5`
2. 距离该热点的上一个信号至少 2 小时（冷却期）
3. 或分数上升至少 `0.5` 个点
4. 或分数首次达到临界阈值（`>=4.5/5`）

**信号上下文**

| 字段 | 内容 |
|-------|---------|
| **为何重要** | "地缘政治热点基于新闻活动、国家不稳定性、地理收敛和军事存在显示显著升级" |
| **可操作洞察** | "提高监控优先级；评估对基础设施、市场和区域稳定的下游影响" |
| **置信度说明** | "由多个数据源加权——新闻（35%）、CII（25%）、地理收敛（25%）、军事（15%）" |

这种多信号方法意味着一个热点升级信号代表跨独立数据流的**佐证证据**，而不仅仅是新闻提及的飙升。

## 区域焦点导航

头部中的 FOCUS 选择器提供到战略区域的即时导航。每个预设都经过校准，以适当的缩放级别居中于该区域的地理范围。

### 可用区域

| 区域 | 覆盖范围 | 主要用例 |
|--------|----------|-------------------|
| **全球** | 完整世界视图 | 概览、跨区域比较 |
| **美洲** | 北美焦点 | 美国监控、NORAD 活动 |
| **欧洲** | 欧盟 + 英国 + 斯堪的纳维亚 + 西俄 | NATO 活动、能源基础设施 |
| **MENA** | 中东 + 北非 | 冲突区、石油基础设施 |
| **亚洲** | 东亚 + 东南亚 | 中国-台湾、朝鲜半岛 |
| **拉丁美洲** | 中美洲 + 南美洲 | 区域不稳定、毒品贩运 |
| **非洲** | 撒哈拉以南非洲 | 冲突区、资源开采 |
| **大洋洲** | 澳大利亚 + 太平洋 | 印太活动 |

### 快速导航

FOCUS 下拉菜单支持快速上下文切换：

1. **突发新闻** - 跳转到受影响区域
2. **区域简报** - 循环浏览各区域以获取态势感知
3. **危机监控** - 锁定特定战区

区域视图编码在可共享的 URL 中，支持直接链接到特定地理上下文。

## 地图固定

默认情况下，地图随页面滚动，允许你向下滚动以查看下方的面板。地图头部中的**固定按钮**可切换粘性行为：

| 状态 | 行为 |
|-------|----------|
| **未固定**（默认） | 地图随页面滚动；向下滚动查看面板 |
| **已固定** | 地图固定在顶部；面板在下方滚动 |

### 何时固定

- **主动监控** - 在阅读新闻面板时保持地图可见
- **交叉引用** - 将地图标记与面板数据进行比较
- **演示** - 在讨论面板内容时展示地图

### 何时取消固定

- **面板聚焦** - 阅读面板时不让地图占用屏幕空间
- **移动端** - 固定功能在移动端被禁用，以更好地利用空间
- **研究** - 专注于数据面板，不受地理干扰

固定状态通过 localStorage 跨会话持久化。
