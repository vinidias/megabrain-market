该目录包含 25 个 MegaBrain Market 智能体技能，面向那些需要聚焦、开箱即用的配方而非完整 OpenAPI 抓取的智能体。每个技能都是一个公开的 `SKILL.md` 文件，包含使用时机指引、认证结构、REST 示例、响应说明以及内容安全处理。

机器可读的权威来源是 [`/.well-known/agent-skills/index.json`](https://megabrain.market/.well-known/agent-skills/index.json)。下方每一行都链接到所提供的配方，其 SHA-256 摘要固定在该清单中。

## 调用模型

当你的客户端能够动态发现工具时，使用 MCP 服务器：

```text
https://megabrain.market/mcp
```

当你想要技能文件中确切的配方调用时，使用 REST：

```text
https://api.megabrain.market
X-MegaBrainMarket-Key: wm_<40-hex>
```

### 国家与战略风险

| `check-country-risk` | 按 ISO 代码获取单个国家的综合国家风险情报——国家不稳定指数（CII）、旅行警示级别和现行制裁敞口。当用户询问某个国家当前风险或不稳定程度时使用。 | [SKILL.md](https://megabrain.market/.well-known/agent-skills/check-country-risk/SKILL.md) |
