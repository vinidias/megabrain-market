## MegaBrain Market（地缘政治）

主变体聚焦于地缘政治情报、军事追踪和基础设施安全监测。

### 核心能力

- **冲突监测** - 实时追踪活跃战区、热点地区和危机区域的升级动态
- **军事情报** - 226 个军事基地、航班追踪、海军船舶监测、兵力集结检测
- **基础设施安全** - 海底电缆、管道、数据中心、网络中断
- **经济情报** - FRED 指标、石油分析、政府支出、制裁追踪
- **自然灾害** - 地震、恶劣天气、NASA EONET 事件（野火、火山、洪水）
- **AI 驱动分析** - 焦点检测（AI 检测到的新闻聚合区域，有别于静态情报热点）、国家不稳定性评分、基础设施级联分析

### 情报面板

| 面板 | 用途 |
|-------|---------|
| **AI Insights** | LLM 综合世界简报，含焦点检测 |
| **AI Strategic Posture** | 战区级军事力量聚合，含打击能力评估 |
| **Country Instability Index** | 31 个一级国家的实时 CII v8 稳定性评分及带符号的 24 小时变动增量 |
| **Strategic Risk Overview** | 综合所有情报模块的综合风险评分 |
| **Infrastructure Cascade** | 电缆、管道和咽喉要道的依赖分析 |
| **Live Intelligence** | 基于 GDELT 的主题信息流（军事、网络、核、制裁） |

### 新闻覆盖

涵盖地缘政治、国防、能源、智库和区域新闻（中东、非洲、拉丁美洲、亚太）的 500+ 精选新闻源。
