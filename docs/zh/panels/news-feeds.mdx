MegaBrainMarket 的每个变体都附有一组**内容流面板** — 区域新闻、主题新闻、科技/创业源、金融和大宗商品流、能源头条和 happy 变体的正能量流。它们共享通用面板框架：排名标题、来源归属、活动追踪和 Cmd+K / 自定义监控器高亮。本页是集合参考，使每个面板的行为可查找，而无需为每个源写一页。

以下表格中的每个面板名称也注册在 `src/config/panels.ts` 的至少一个 `*_PANELS` 块中。当你打开一个面板且内容为空时，请检查 [数据来源](/zh/data-sources) 和全局 [健康端点](/zh/health-endpoints) 了解底层源的状态。

关于这些源共享的评分、新鲜度、去重、故事追踪、摘要过滤、冷却、LLM 和偏见控制，请参见权威的
[新闻摘要与简报方法论](/zh/methodology/news-digest-and-briefing)。

## 区域新闻

区域面板将世界限定为一个大区的新闻流，语言覆盖和来源精选针对该区域优化。它们在 geopolitical 变体中共享 `regional news` 族，并出现在 Cmd+K 的 "News" 组中。

| 面板 id | 标题 | 变体 |
|---|---|---|
| `politics` | 世界新闻 | Full / tech / finance / commodity |
| `us` | 美国 | Full |
| `europe` | 欧洲 | Full |
| `middleeast` | 中东 | Full |
| `africa` | 非洲 | Full |
| `latam` | 拉丁美洲 | Full |
| `asia` | 亚太 | Full |

全部免费。来源精选和权限位于 `src/config/variants/full.ts`（及等价文件）。

## 所有新闻面板的共享行为

- **活动追踪** — 新到达的条目获得标准的新条目视觉效果。
- **自定义监控器高亮** — 匹配的文章以 [监控器](/zh/panels/monitors) 颜色作为强调色展示（包括聚类内）。
- **Cmd+K 可发现性** — 每个面板可通过其标题和类别搜索。
- **默认免费** — 没有新闻源面板是 PRO 门控的。

### 支撑端点

| 源族 | 端点 | 备注 |
|---|---|---|
| 摘要源 | `GET /api/news/v1/list-feed-digest` | 按站点变体预聚合的 RSS 摘要。 |
| 摘要缓存查询 | `GET /api/news/v1/summarize-article-cache` | 用于确定性文章摘要键的仅缓存 GET；使用 `POST /api/news/v1/summarize-article` 生成。 |
| arXiv 论文 | `GET /api/research/v1/list-arxiv-papers` | ResearchService 学术论文源。 |
| 热门仓库 | `GET /api/research/v1/list-trending-repos` | ResearchService GitHub 热门源。 |
| Hacker News 条目 | `GET /api/research/v1/list-hackernews-items` | ResearchService Hacker News 源。 |
| 捐赠活动 | `GET /api/giving/v1/get-giving-summary` | happy 变体捐赠面使用的全球个人捐赠与公益摘要。 |
