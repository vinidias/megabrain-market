**Daily Market Brief** 面板（内部 id `daily-market-brief`）是 [Latest Brief](/zh/panels/latest-brief) 面板的市场聚焦兄弟面板。它不产出世界情报摘要，而是生成对当日市场的态度化、主题化解读 — 每个条目标记为 `Bullish`、`Defensive` 或 `Neutral`，附带来源归属和框架上下文。

## 面板显示内容

按主题分组的简报条目列表。每个条目显示：

- **态度标签** — `Bullish` / `Defensive` / `Neutral`（来自 `src/components/DailyMarketBriefPanel.ts` 中的 `stanceLabel()`）。
- **叙述** — 简报对该主题的评论。
- **来源标签** — 生成该条目的信号的归属信息。
- **新鲜度** — 面板顶部页脚使用 `describeFreshness()` 显示底层简报的新旧程度，以及渲染来自 `live` 获取还是 `cached` 路径（类型为 `BriefSource = 'live' | 'cached'`）。

面板顶部的 **框架选择器** 允许 PRO 用户在不同分析框架下重新生成简报（仅限客户端生成的分析；说明字符串中对此有明确说明）。

面板 id 为 `daily-market-brief`；规范组件为 `src/components/DailyMarketBriefPanel.ts`。标题见 `src/config/panels.ts:51`。

## 如何访问

- **Cmd+K**：输入 *brief* 或 *market brief*。
- **按变体的可用性**：在 **full/geopolitical** 和 **finance** 变体中注册并默认启用，两者均为 `premium: 'locked'`。在 tech、commodity 或 happy 变体中不存在。

## 数据来源

由预测管道生成，通过 `@/services/daily-market-brief` 读取。底层信息源由当日的市场信号（报价、大宗商品价格、宏观信号、板块走势、预测市场）组合而成。发布时间戳通过 `formatGeneratedTime()` 在面板所有者的时区中渲染。

关于相邻的 MegaBrainMarket Brief 的来源过滤、去重、LLM 接地与偏差立场，请参见 [News Digest and Briefing Methodology](/zh/methodology/news-digest-and-briefing)。

## 刷新频率

每日频率，作为预测 cron 的一部分刷新。面板通过 `describeFreshness()` 展示，让读者可以直接看到当前简报的新旧程度；过期获取会将来源徽章显示为 `cached`。

## 层级与门控

**PRO（硬锁定）。** `src/config/panels.ts:51` 中的 `premium: 'locked'`。当存在有效的 `MEGABRAIN_MARKET_API_KEY` **或** `isPro` 为 true 时，`isPanelEntitled` 解锁该面板（`daily-market-brief` 在 `src/config/panels.ts:973` 的 `apiKeyPanels` 允许列表中）。
